ETL Pipeline for Business Data Integration


As part of an academic project carried out in pairs (30 hours), I designed and deployed a complete ETL pipeline designed to feed a BI warehouse with open data from the SIRENE database. The project began with automated data extraction via REST API and web scraping (Selenium, BeautifulSoup), followed by extensive cleansing, normalization and enrichment work, notably by geocoding addresses using the data.gouv.fr address API.


The data was then modeled, structured and integrated into a CSV format warehouse that could be used with Pandas. A statistical analysis was carried out to highlight key indicators (geographical distribution, types of activity, company size, etc.), summarized in a detailed report. The aim of this project was to simulate a real-life case of feeding a decision support system for public or private stakeholders.
